Introduction to CSS — What is CSS, why we use it

Inline CSS is a way of adding CSS directly inside an HTML element using the style attribute. It applies styles only to that specific element.

Features

  • Affects only that element.
  • Has the highest priority.
  • Hard to maintain for large websites.

CSS Syntax — Selector, Property, Value

CSS is written in the form of rules that tell the browser how to style HTML elements. A CSS rule has three main parts:

  1. Selector – the HTML element you want to style.
  2. Property – the style attribute you want to change.
  3. Value – the setting for that property
